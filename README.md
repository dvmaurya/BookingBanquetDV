# BookingBanquetDV

## 📌 Project Overview

BookingBanquetDV is a desktop application built using **WPF (.NET)** for managing banquet bookings efficiently.
The application provides a user-friendly interface to handle booking operations, customer details, and scheduling.

---

## 🚀 Features

* 📅 Banquet booking management
* 👤 Customer information handling
* 🎨 Modern UI using WPF (XAML)
* 🔄 Dynamic UI with styles and triggers
* 🧩 Structured using MVVM architecture

---

## 🛠️ Technologies Used

* .NET (WPF)
* C#
* XAML
* MVVM Design Pattern

---

## 📂 Project Structure

* **Views** – UI screens (XAML)
* **ViewModels** – Business logic and bindings
* **Models** – Data structures
* **Resources** – Styles, themes

---

## ▶️ How to Run

1. Clone the repository
2. Open the `.sln` file in Visual Studio
3. Build the solution
4. Run the application
